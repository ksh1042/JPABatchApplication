# JPABatchApplication

## 개요 및 목적
- JPA와 Hibernate를 이용한 간단한 사용자 업데이트 시스템 구현
- 테이블 형식의 파일(CSV, EXCEL)와 같은 파일을 읽어들여 데이터베이스로 입력하는 JPA Application

## 특징
- 개인용 온프레미스 환경 Oracle DB 서버 연동
